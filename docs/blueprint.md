# **App Name**: Gym Journey

## Core Features:

- Workout Schedule Builder: Create and edit a weekly workout routine by assigning muscle groups to each day.
- Exercise Management: Add exercises to each workout day, selecting from a preloaded database or creating custom exercises, grouped by muscle.
- Workout Logging: Log workout details (sets, reps, weight) for each exercise and store workout history.
- Progress Tracking: Track progress over time for weight lifted and total volume, showing graphs for strength increase per exercise.
- Body Data Input: Input and update body weight, height, and age (optional) for tracking changes over time.
- Growth Prediction System: Estimate strength progression and muscle growth trends based on workout consistency, progressive overload, body weight changes, and training volume, displaying as estimated strength increase and general muscle growth trend. Note: This is an estimate only, not medical advice. The AI will use training data to tool to predict possible growth
- Daily workout reminder: Set up daily workout reminder

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for a sense of strength and determination.
- Background color: Very light lavender (#F0F8FF), almost white, to ensure readability and focus.
- Accent color: Electric violet (#8F00FF) for interactive elements and highlights, providing a modern edge.
- Body and headline font: 'PT Sans' (sans-serif) for a clean, modern aesthetic and good readability.
- Use consistent, minimalist icons to represent exercises and muscle groups.
- Employ a card-based layout for exercises and routines for easy organization.
- Use subtle transitions and animations to provide feedback on user actions, like logging workouts.
